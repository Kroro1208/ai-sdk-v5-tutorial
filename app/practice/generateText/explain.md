# 基本的な実装

`generateText()` はLLMからテキストを一括生成する同期関数です。`streamText()`と異なり、生成が完了するまで待機してから結果を返すため、バッチ処理や自動化タスクに適しています。

```ts
import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

const result = await generateText({
  model: openai("gpt-4o"),
  system: "あなたはフレンドリーなアシスタントです。",
  prompt: "今日の天気はどうですか？",
});

console.log(result.text); // 生成されたテキスト
console.log(result.usage); // トークン使用量
```

## 主な特徴

`generateText()`の主な特徴

- **完全なレスポンス**: LLMからの応答を最後まで受け取ってから結果を返す
- **シンプルな使用法**: ストリーミング処理を気にせず、完成したテキストを直接取得
- **詳細情報**: 生成されたテキストに加えて、トークン使用量などのメタデータも取得
- **使用用途**: 翻訳、要約、Q&Aなど、完全な結果が一度に必要な場合に最適

## streamText() との使い分け

| 特徴         | generateText()       | streamText()             |
| ------------ | -------------------- | ------------------------ |
| 処理方式     | 同期（完了まで待機） | 非同期（ストリーミング） |
| 用途         | バッチ処理、自動化   | リアルタイム表示         |
| レスポンス   | 完全なテキスト       | チャンク単位             |
| ユーザー体験 | 待機時間あり         | 即座に表示開始           |
